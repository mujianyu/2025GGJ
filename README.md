# 2025GGJ

## 1. 环境配置

- unity 版本：2022.1.9f1
- Spine 版本：4.2
- 分辨率：3840x2160

## 3.项目文件介绍

- Animation:动画
- Font: 字体
- Image: （GameEnd：结束动画，Designer：制作者名单，PlayRule：游戏规则，Setting：游戏设置，HomePage：主页，MainUI：游戏时的背景）
- Music：游戏音效
- Scenes: 游戏场景
- Scripts: 游戏脚本
- Spine: Spine插件
